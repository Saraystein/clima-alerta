O que é o Conexão Climática?

O Conexão Climática é uma plataforma inteligente de consulta do tempo, projetada para estudantes, professores e qualquer pessoa que deseje se preparar melhor para as mudanças climáticas no dia a dia.

Objetivo

O Conexão Climática tem como missão facilitar o acesso à previsão do tempo de forma prática, moderna e visualmente amigável. O site reúne dados atualizados de clima, gráficos interativos, curiosidades meteorológicas para ajudar usuários a se prepararem melhor para o dia.

Tecnologias Utilizadas
  • 🌐 HTML, CSS, JS (Frontend)
  • 🎨 Bootstrap 5 para layout responsivo
  • 🧠 Flask (Python) no Backend
  • ☁️ API OpenWeatherMap para dados meteorológicos
  • 📊 Chart.js para gráficos futuros
